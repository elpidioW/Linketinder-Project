# LinkeTinder

Sistema de recrutamento inspirado em plataformas como LinkedIn e Tinder, de modo a facilitar a contratação de um candidato às cegas por meio de "curtidas", gerando matches quando há interesse mútuo.

Atualmente a aplicação permite o cadastro e listagem de candidatos e empresas no backend e no frontend.
Front e back ainda não se comunicam.
No frontend há persistência de dados utilizando LocalStorage.



### Feito por: Wallace Elpidio Pereira cardoso <br><br>


Para executar backend:
Necessário Java JDK 8+ e Groovy instalado
Dentro da pasta do projeto, execute: groovy src/Main.groovy
